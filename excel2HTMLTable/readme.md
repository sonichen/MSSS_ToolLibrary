# excel表格转前端表格

需求：excel表格转前端表格(按照官网样式)，具体需要根据excel修改

该份代码处理的列名是“部门”，“职位”，“姓名”

技术：Java

备注：

依赖

```xml
      <dependency>
            <groupId>org.apache.poi</groupId>
            <artifactId>poi</artifactId>
            <version>5.0.0</version>  
        </dependency>
        <dependency>
            <groupId>org.apache.poi</groupId>
            <artifactId>poi-ooxml</artifactId>
            <version>5.0.0</version>  
        </dependency>
```
